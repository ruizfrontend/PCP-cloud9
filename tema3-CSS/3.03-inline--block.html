Un párrafo es por defecto un elemento block. Crea un salto de línea antes y otro después.

Lo habitual es que dentro de un párrafo todos los elementos sean inline como esta imagen: que se alinea siguiendo el baseline del texto: la linea imaginaria sobre la que se apoyan todas las letras., pero podemos cambiarlo mediante vertical align . Esto hace que el alto de la linea en que se encuentra se ajuste al tamaño del elemento mayor.

Los Elementos block como este div se comportan como cajas independientes

Se ajustan al tamaños de sus contenidos, pero permiten dimensionarlos
Y esto puede hacer que parte del contenido se salga de su caja.
Y hacer que 'floten' uno al lado del otro

Este párrafo, al estar al lado de elementos flotantes ajusta su espacio a estos.

Pero en este otro, mediante clear, no permitimos que se ajuste a los elementos flotantes.
